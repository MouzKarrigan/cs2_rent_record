# CS饰品租赁收益管理系统

一个基于 Vue3 + ElementPlus 的纯前端饰品租赁收益管理系统，用于记录和计算CS2饰品的租赁收益情况。

## 功能特性

### 核心功能

1. **饰品管理**
   - 添加饰品记录（名称、购入价格、购入日期）
   - 查看所有饰品列表
   - 展开/收起饰品详情

2. **租赁记录管理**
   - 为每个饰品添加租赁记录
   - 输入日租金和手续费率
   - 选择出租时间段，自动计算出租天数
   - 查看租赁记录列表（每页10条，支持分页）
   - 删除单条租赁记录

3. **收益计算**
   - 自动计算每条租赁的总租金
   - 自动扣除手续费，计算净收益
   - 计算单个饰品的总租金收入
   - 计算单个饰品的收益率和年化收益率
   - 统计所有饰品的总购入价和总租金
   - 计算整体收益率和总年化收益率

4. **数据导入导出**
   - 导出所有数据为JSON格式文件
   - 从JSON文件导入数据（会覆盖现有数据）

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI组件库**: Element Plus
- **构建工具**: Vite
- **语言**: JavaScript

## 使用说明

### 1. 添加饰品

- 点击左上角"添加饰品"按钮
- 填写饰品名称、购入价格、购入日期
- 点击"确定"完成添加

### 2. 添加租赁记录

- 点击饰品行右侧的"添加租赁记录"按钮
- 输入日租金（人民币，保留两位小数）
- 输入手续费率（百分比，保留两位小数）
- 选择出租时间段（系统会自动计算天数）
- 点击"确定"完成添加

### 3. 查看租赁详情

- 点击饰品行左侧的下箭头展开租赁记录
- 查看该饰品的所有租赁记录
- 如果记录超过10条，使用分页导航查看

### 4. 删除租赁记录

- 在展开的租赁记录列表中
- 点击要删除的记录右侧的"删除"按钮

### 5. 导出数据

- 点击右上角"导出"按钮
- 系统会自动下载JSON格式的数据文件

### 6. 导入数据

- 点击右上角"导入"按钮
- 选择之前导出的JSON文件
- 确认后会覆盖当前所有数据

## 数据计算说明

### 收益率
```
收益率 = (总租金收入 / 购入价格) × 100%
```

### 年化收益率
```
年化收益率 = (总租金收入 / 购入价格 / 持有天数 × 365) × 100%
```

### 总年化收益率
使用加权平均计算，权重为各饰品的购入价格。

## 部署方式

### 方式1: 静态网站托管

1. 构建项目：
   ```bash
   npm run build
   ```

2. 将 `dist` 目录下的所有文件部署到任何静态网站托管服务：
   - GitHub Pages
   - Netlify
   - Vercel
   - 阿里云OSS
   - 腾讯云COS
   - 或任何支持静态文件的Web服务器

### 方式2: 本地运行

1. 克隆或下载项目
2. 安装依赖：`npm install`
3. 运行开发服务器：`npm run dev`
4. 在浏览器中访问显示的地址

### 方式3: 服务器部署

1. 构建项目：`npm run build`
2. 将 `dist` 目录复制到服务器
3. 配置Nginx或Apache指向该目录
4. 确保所有路由都指向 `index.html`

#### Nginx配置示例：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 数据存储说明

- 本系统为纯前端应用，数据仅存储在浏览器内存中
- 刷新页面后数据会丢失
- 请定期使用"导出"功能备份数据
- 下次使用时通过"导入"功能恢复数据
